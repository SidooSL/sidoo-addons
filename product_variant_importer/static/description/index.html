Product Variant Importer

The user can import a Excel file with products, variants and stock

The structure of the excel should be like this:

  • First sheet products
  • Second sheet variants
  • Third sheet stock

Notable features:

  1. Allow to mark import stock field

    When mark the import stock field the wizard will import the stock sheet in excel file

  2. Allow to avoid import stock quantities

    In order to check if import is correct you can go to debug mode and mark the import without qty field, to avoid import stock quantities

Usage

To use the functionality

  1. Go to Inventory / Settings / Wizard Import Products
  2. Upload your excel file with required structure
  3. Mark the import stock field if you want to import the stock sheet
  4. Go to debug mode and mark import without qty to test the product creation and variants, with this check you can delete the products if there is any problem
  5. When everything is correct, use the wizard without this field to import the stock quantities.
